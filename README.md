# Manufacturing-Defects-ML-Model-and-Power-BI-Analysis
This project combines machine learning and business intelligence to analyse manufacturing defect patterns and predict defect risk using operational production metrics. A classification model built in Python generates defect probability scores which are integrated into a Power BI dashboard to support operational quality monitoring and risk-based decision making.
